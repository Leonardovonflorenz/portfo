About The Project

This project simulates the creation of a Dashboard from a company information, it is based on the Database that is shipped along with Microsoft Access and should work as a first approach example of the typical questions that procurement controller should help to solve. It starts with a general view of Spend and drilling down to the base of the information what typically is an order.
